Technical Writing

Technical Writing Course, Missouri S&T

Technical Writing Document Design Data Visualization

Overview

A professionally formatted single-page product specification sheet written for the Game Boy Advance. The document was designed to communicate both marketing-oriented product descriptions and precise technical hardware details to a mixed audience, following standard product spec sheet conventions.

Document Highlights

  • Product description covering the GBA's key improvements over the Game Boy Color, including the 32-bit architecture, expanded color palette, and horizontal form factor.
  • Annotated feature diagram labeling all hardware components of the console.
  • Historical sales chart (2001–2009) presenting unit sales data in a visual format.
  • Technical specifications table covering CPU, memory, display, physical dimensions, and battery life.
