Embedded Systems – Simon Board Project 2

Sep 2024 – Oct 2024

Assembly & C | ATmega324pb (Simon Board) | Microchip Studio

Overview

Utilized assembly and C to control LEDs on an ATmega324pb "Simon Board" microcontroller. The project involved converting existing assembly code into C, then designing and programming a functional down counter from scratch — covering real-time timer configuration, interrupt service routines, and cross-language debugging as a team.

My Contributions

  • Created the updated visual flow charts for both the code conversion and down counter programs.
  • Helped translate the main function logic from Project 1 assembly into C.
  • Set up the general program structure for the Down Counter, including port initialization and the main polling loop.

Technologies Used

Assembly C AVR / ATmega324pb Microchip Studio Interrupts Timers
